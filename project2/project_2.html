Motivation

It is a common topic nowadays to find controversy regarding whether there is climate change or not.
For the ones that do not know and according to nasa.com, " climate change is a change in the usual weather found in a place.
This could be a change in a place's usual temperature for a month or season.
Or it could be a change in how much rain a place usually gets in a year etc.".

Disregarding of the different opinions that are hold concerning this topic
(people in favour, people against and people that do not care), it can be observed that something out of the normal
is happening in our planet .
It is our duty, as global citizens, to explore objective datasets in order to check if climate change is true
and if so, investigate the consequences.





Method

The file selected was taken from: data.giss.nasa.gov: Kobenhavn.
This file contains monthly temperatures of Copenhagen (Denmark), from the years 1880 to 2017;
The detailed file information is the following:

  • 1st Column:     Year.
  • 2nd-13th Column:   Months from January to December.
  • 14th Column:   D-J-F (Average of December, January and February)
  • 15th Column:   M-A-M (Average of March, April and May)
  • 16th Column:   J-J-A (Average of June, July and August)
  • 17th Column:   S-O-N (Average of September, October and November)
  • 18th Column:   metANN (Average of all the months)

When reading the file it is observed that from 2012 to 2017 there are some temperature outliers that surely
are errors in the data gathering (i.e. 999.9 degrees is not possible, especially in Copenhagen!).
For the visualization performed in this assignment we have used the dataset file in both a .txt file and a .csv.
The .csv file was obtained with the following python code:

    list_of_lines = [] # Initialises the list that will contain all the modified lines of our file
    with open('temperaturesKobenhavn.txt', 'r') as tsv: # Loads our file
        for line in tsv: # Go line by line of our file
            new_line = ",".join(line.split()) # Splits the elements of each line by white space,and joins the elements by ","
            list_of_lines.append(new_line) # Append each modified line to the list.
    txtfile = open("temperaturesKobenhavn.csv", 'w') # here I save each line as a .csv file
    for element in list_of_lines:
        txtfile.write(element+"\n") # Add each line on a new line of the .csv file
    txtfile.close()

We decided to make two plots:
The first one is a barplot whose x axis and y axis are the even years (1880-2011)
and the annual average of these years(obtained from the last column of the dataset) respectively;
this temperature is useful because it helps us to see the trend of the annual mean temperature over the years.
The second plot is a heatmap of all the values in the dataset (the x axis are the column names,
and the y axis are the years from 1880 (up) to 2011 (down), this is useful because in a quick look we can
check if there are clusters of a certain temperature or a temperature difference in a specific year-month period.

Discussion

IMPORTANT NOTE: When plotting the barplot we observe a peak on the left that we do not know where it comes from.
In the barplot we observe that from the 1990's onwards some of the temperatures reached are higher than normal. However, we do not
observe a clear fluctuation trend over a 10 year period.
In the heatmap we observe that monthly temperatures genaraly do not differ in the period 1880-2011.
Nevertheless we do see that temperatures per month seem to follow each other for about 3 consecutive years.

Conclusion

A two plot analysis is not enough to infer that there is climate change going on.
Further analysis should be performed.
